# AF-LNA
ADA4522-2ARMZ を 4パラで使用した低周波用のローノイズアンプです。  
帯域は 0.1～1000Hz、ゲインは 60dB、ノイズは 5nV/√Hz 以下となるように設計しています。  
⇒ 実測で 3nV/√Hz 程度となりました
